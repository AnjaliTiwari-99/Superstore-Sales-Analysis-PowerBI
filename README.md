# Superstore Sales Analysis Dashboard

This project is an interactive **Power BI dashboard** built to analyze sales and profit performance of a retail superstore.  
The dashboard helps understand **sales trends, product performance, and regional demand**, enabling better business decision-making.

---

##  Project Overview

The goal of this project is to explore and analyze retail sales data to identify important business insights such as:

- Which product categories generate the most revenue
- Which categories contribute the highest profit
- How sales vary across different regions and states
- Which products are most frequently ordered

Using Power BI, the data is transformed into an interactive dashboard that allows users to explore the data easily.

---

##  Key Insights

- **Total Sales:** $1.57M  
- **Total Profit:** $175.3K  
- **Total Orders:** 6K+  
- **Total Quantity Sold:** 22K  

Some important findings from the analysis:

- **Technology** is the most profitable category, contributing **51.6% of total profit**
- **Furniture** has the lowest profit share despite high sales
- Certain states like **California and New York** generate the highest revenue
- Products such as **Phones and Chairs** contribute significantly to overall sales

##  Dashboard Features

The dashboard provides:

- Sales and profit overview
- Category and sub-category performance analysis
- State-wise sales analysis
- Order distribution across products
- Interactive filters for:
  - Region
  - State
  - City
  - Product Category

These filters help users explore the data and uncover deeper insights.

---

##  How to Use

1. Download the `.pbix` file from this repository
2. Open it using **Power BI Desktop**
3. Use the filters and visuals to explore sales and profit trends
